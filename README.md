# Prompt Power Privacy Policy

Last updated: January 2026

Prompt Power is a Chrome extension designed to help users improve the clarity and structure of AI prompts directly within the browser.

---

## Data Collection

Prompt Power does not collect, store, transmit, or sell personal or sensitive user data.

---

## User Input and Processing

Any text entered by the user is processed locally in the browser or sent directly to the AI service explicitly chosen by the user (Bring Your Own Key). Prompt Power does not log, retain, or access this data.

---

## Data Usage and Purpose

Prompt Power only processes user input for the sole purpose of improving prompt clarity at the user’s request. No data is used for profiling, advertising, analytics, or any secondary purpose.

---

## Accounts and Authentication

Prompt Power does not require user accounts or authentication.

---

## Third-Party Services

Prompt Power does not use third-party analytics, advertising, tracking, or data collection services.

---

## Data Sharing

Prompt Power does not share user data with the developer or any third parties.

---

## Security

All processing occurs within the user’s browser environment or directly between the user and their chosen AI provider.

---

## Changes to This Policy

This privacy policy may be updated as the extension evolves. Any changes will be reflected on this page.

---

## Compliance Statement

Prompt Power complies with the Chrome Web Store Developer Program Policies, including the User Data Privacy requirements. The extension does not collect, store, sell, or share personal or sensitive user data.
