Prompt Power Privacy Policy

Last updated: January 2026

Prompt Power is a Chrome extension designed to help users improve the clarity and structure of AI prompts directly in the browser.

Data Collection

Prompt Power does not collect, transmit, store, or sell personal or sensitive user data.

User Input

Any text entered by the user is processed locally in the browser or sent directly to the AI service explicitly chosen by the user. Prompt Power does not log, retain, or access this data.

Accounts

Prompt Power does not require user accounts or authentication.

Third-Party Services

Prompt Power does not use third-party analytics, advertising, or tracking services.

Data Sharing

No user data is shared with the developer or any third parties.

Security

All processing occurs within the user's browser environment or directly between the user and their chosen AI provider.

Changes

This privacy policy may be updated as the extension evolves. Any changes will be reflected on this page.

Contact

If you have questions about this privacy policy, you can contact the developer via the Chrome Web Store listing.
